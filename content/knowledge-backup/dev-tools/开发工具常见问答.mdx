# 开发工具常见问答

- **GSC 索引未更新**：删除页面后需返回 404，再提交新 sitemap 并在 GSC 手动请求移除，2–4 周自动刷新。
- **Dia 界面英文**：安装沉浸式翻译插件即可；Sky 同学实测成功。
- **Monica 是否等于 Claude Code？**：不是，Monica 等套壳只是 Claude 网页版；真正的开发能力需 Claude Code。
- **有 Codex 还需要 Claude 吗？**：需要，Codex 逻辑强但速度慢；当你感受到 Codex 不够时再切回 Claude 4.5。
- **Windows 能否完成课程？**：可以，环境配置比 Mac 繁琐一些；推荐豆包浏览器 + Cursor + VS Code。
- **快捷查询技巧**：群内搜索 `#举手`、`助教`、工具名、`破冰`，快速定位历史讨论；提问时先写清楚场景再贴截图。
