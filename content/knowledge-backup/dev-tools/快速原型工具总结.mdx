# 快速原型工具总结

### Bolt.new
- **额度与费用**：免费版每日 5 次对话；付费版闲鱼年卡约 203 元。官方额度逐渐收紧，新号可体验流程但不宜依赖。
- **使用建议**：提前写好提示词，一次生成一个简单前端页面 3–5 天够用；下载代码后交给 Cursor 精修。Vercel 部署失败时手动上传 GitHub。
- **常见问题**：
  - 登录失败 `Failed to authenticate` → 切换到美国节点、开启 TUN 或清缓存。
  - Token 用完 → 分步拆需求或改用 V0。
  - 免费额度提示耗尽 → 新建账号或充值。
  - 多轮迭代超过 v9 效果急速下降。

### V0.dev（Vercel 出品）
- **优势**：界面与代码质量高、生成速度快、免费额度相对宽松；Next.js 表现优于 Bolt。
- **导出方法**：右上角 `Code → Export`，或点击小猫图标同步到 GitHub；导出后在 Cursor 中 `npm install` / `npm run dev`。
- **价格**：正式套餐 30 美元/月，闲鱼一年账号 200 元左右。
- **策略**：注册即送 5 美元额度，足够多次迭代。新手期做 UI，后期仍需 Cursor/Claude Code 打磨。

### Lovable
- **定位**：UI 设计最强，页面美观度普遍优于 V0/Bolt（学员松果案例 `https://bubble-body-talk.lovable.app/`）。
- **限制**：不支持 Next.js；适合 React 项目。先做好界面，再在 Cursor/Claude Code 中补全逻辑。

### 工具横向对比
| 工具 | 优势 | 劣势 | 结论 |
| --- | --- | --- | --- |
| V0 | 质量高、速度快、可直接部署 | 免费额度有限，持续需付费 | 快速原型首选 |
| Bolt | 上手简单、界面直观 | 免费额度极少，复杂需求易失败 | 体验流程即可 |
| Lovable | UI 漂亮、易用 | 不支持 Next.js，需要后续打磨 | 用于视觉设计 |
| Stitch / Orchids / Trickle / Figma Make | 可与 Bolt/V0 替换互补 | 功能各异需探索 | 探索阶段可尝试 |
