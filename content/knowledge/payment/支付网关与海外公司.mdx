# 支付网关与海外公司

### 网关选择优先级

- **Creem**：对中国用户最友好，可个人入驻；审核严格，适合准备充分的新项目。

- **Paddle**：审核相对宽松，适合传统 SaaS 或非 AI 产品。

- **Stripe**：功能最强，但需香港银行卡或海外公司；企业方案更正规。

- **备用**：LemonSqueezy、PayPal（门槛高、手续费高，仅作补充）。

### Creem 申请重点
- 必备条件：上线可用的网站、真实功能、内容合规。
- 审核策略：官方会刻意输入敏感词（色情、违法等），需提前做好 Prompt 过滤与内容审核。
- 流程：
  1. 准备产品官网、说明、使用文档、隐私/退款政策等。
  2. 提交申请，必要时提供测试账号。

> **Note (info):** 3. 若被拒，可在完善过滤后再次申诉，但注意“同一产品被拒=账号冻结 3 个月”，首提务必准备充分。

> **Note (warning):** - 成功案例：AI 游戏站通过多家网关审核；失败案例多出现于 AI 图片生成等高风险场景。

### Paddle & LemonSqueezy
- Paddle 审核较快，手续费略高；适合 Creem 被拒后的备用方案。
- LemonSqueezy 支持一次性与订阅收费，可作为备用收款通道。

### Stripe + 海外公司
- **香港银行卡个人户**：拥有香港银行卡（如汇丰 One、众安）即可快速开通 Stripe 个人账户，审核数小时即可完成。
- **英国公司路径**：
  - 费用：首年约 1200 元（含地址托管），次年年审 859 元 + 地址托管 400 元 + 零申报 599 元。
  - 英国公司海外运营可合法零申报，税务压力小。
  - 注册流程：准备护照、地址证明、公司名称等 → 委托服务商（淘宝可选） → 1–2 天出证照。

> **Note (info):** - **美国/香港公司**：手续更复杂、费用更高，一般不推荐。

> **Note (warning):** - 风控提醒：Stripe 会关注高风险业务，需准备清晰的业务说明与合规材料，资金尽量分散提现。

> **Note (info):** ### ⚡ PayPal（不推荐）
- 需营业执照或个体户资质，审核 1–2 周。双重短信验证必须用真实手机号，接码平台易失败。
- 手续费高、提现麻烦、争议处理偏向消费者。仅在用户必须 PayPal 支付或其他网关被拒时作为备选。

### Wise 海外账户
- 英国线上银行，可快速开通多币种账户与虚拟卡，费用低。
- 适用场景：接收海外客户款项、支付 API 费用。
- 开户流程参见官方文档或小红书教程，实名认证后即可使用。
