# 工具选型路线图

- **阶段性思路**：前 30 天用 Cursor 建立基础和项目意识，搭配 Codex 辅助；熟练后切换到 Claude Code + Codex 组合（助教小明的路径）。
- **核心组合**：`Cursor（IDE）` + `Claude/Codex（大模型）` + `Dia/Comet（调研浏览器）` + `NotebookLM、沉浸式翻译` 等效率工具。
- **国产备选**：GLM-4.6 Lite/Pro、豆包 TRAE、Augment Code 等都可作为 Claude Code 的临时替代方案；预算有限先上国产，关键项目再回到 Claude 4.5。
- **常见疑问**：
  - “已经用 VS Code 了还要换 Cursor 吗？” → 要换，操作体验几乎相同，但 Cursor 原生集成 AI。
  - “Codex 和 Claude Code 差别大吗？” → Claude Code（Sonnet 4.5）逻辑最强，Codex 稍慢但稳定且便宜，两者互补。
  - “我应该先买哪款？” → 先保证稳定的工具链，再考虑省钱：GPT-4/Cursor Pro/中转服务为基础，随后再看 Claude Pro 或国产付费。
