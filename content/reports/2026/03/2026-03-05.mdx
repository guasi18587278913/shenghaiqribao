### 💡 今日群聊精华：OpenClaw的“一人军队”开发流、反代封号潮与豆包专家模式崛起

**📅 日期：** 2026-03-05

本期讨论的核心聚焦在现象级开源项目 OpenClaw（代号“小龙虾”）背后的全自动开发流程复刻，以及多模态数据抓取工具 Agent Reach 的实战应用。与此同时，关于“反代（反重力）”账号的大规模封禁引发了由于低成本方案失效带来的焦虑与对策讨论，国产大模型在特定场景下的表现也收到了不少好评。

***

### 🚀 技术焦点：OpenClaw 与“一人军队”开发流

本周 OpenClaw 项目 Star 数破 5k，其极高的迭代速度引发了群友对“AI 辅助开发”极限的讨论。

* **“小龙虾”的开发秘密：**
* **全自动流水线：** 核心维护者（小龙虾）并非单纯手动撸代码，而是构建了一套 AI 闭环。利用 Opus 4.6 等模型，配合 **队列循环任务模式**。
* **AI Review 机制：** 并非人工逐行审核，而是建立了 Review Agent。当 PR 提交后，AI 会根据预设规则（如 diff 分析、静态分析、测试覆盖率检查）自动审核，甚至自动动态创建 Agent 并行处理。
* **高频提交真相：** 上百个 Commit 往往是由 AI 拆解任务列表后逐步完成的，人工仅需在关键节点做二次确认（或仅看 summary）。
* **如何复刻这套流程？**
* **核心思路：** Issue -> AI 开发 -> AI 提交 (Commit) -> AI 提单 (PR) -> AI 审核 (Review)。
* **实战配置（群友“宇宙”分享）：** 可以在腾讯云服务器上使用 Tmux + OpenClaw 搭建“多智能体工作室”。
* **窗口分工：** 0号指挥官（总控/飞书对接）、1号核心开发（Qwen-Coder）、2号视觉验收（Kimi/Qwen-plus）、3号运行时环境。
* **闭环验证：** 通过飞书机器人发送指令，监测 tmux 会话状态，实现远程遥控。

### 🛠️ 工具测评：Agent Reach 与数据抓取

群友对 **Agent Reach** 给出了极高评价，认为是目前获取全网数据的利器，但也存在明显的风控隐患。

* **功能亮点：**
* **全能抓取：** 支持将任意网页转为 Markdown（Jina Reader），无 Key 提取 YouTube/B站 元数据与字幕，甚至能通过模拟登录处理 LinkedIn 和 Boss直聘的数据。
* **绕过限制：** 利用 cookie 模拟登录状态，绕过 Twitter/X 的官方付费 API 限制。
* **高危预警：**
* **封号风险：** 小红书（RedNote）和 Twitter（X）的方案存在极高的封号风险，**严禁高频使用**。
* **替代方案：** 针对 Twitter 建议尝试 Grok 4.1 fast；针对抖音建议尝试 seed 2.0。

### ⚠️ 避坑指南：“反重力”封号潮与模型选择

关于使用“反代”（Reverse Proxy，群内常称“反重力/Antigravity”）来白嫖或低成本使用 Gemini/Claude 服务的讨论非常热烈。

* **现状：全线封禁**
* 无论是自建反代还是使用第三方工具（如 Antigravity tools），近期频繁出现 **403 错误**。
* **结论：** 反代方案目前非常不稳，账号“七进七出”（封了又解，解了又封），不建议用于生产环境。
* **Claude Code (CC)：** 官方对于 GitHub 核心维护者（>5000 star或高活跃度）提供半年 $200 的免费额度，建议符合条件的去申请正版，不要碰反代。
* **模型红黑榜：**
* **豆包（字节跳动）：** “专家模式”备受好评。群友实测认为在自媒体方案、深度研究方面，比 DeepSeek (DS) 的“深度思考”更靠谱，且搜索抖音数据的能力极强。
* **GLM-5 (智谱)：** Coding 模型抢手，网页端 Token 购买非常困难，需要利用脚本或高频刷新技巧。
* **Cursor 替代：** 由于 Cursor 薅羊毛变难，群友推荐 Codex 或闲鱼购买 Team 账号作为平替。

### 📝 总结与行动建议

1. **拥抱自动化开发：** “小龙虾”模式证明了 AI Agent 已经能承担代码 Review 和日常 Commit 的重任，开发者应尝试将工作流从“写代码”转向“设计 Agent 协作流”。
1. **放弃反代幻想：** 针对 Gemini 和 Claude 的“反重力”薅羊毛时代基本结束，封号成本过高。生产环境请转向官方 API、Azure 通道或国内稳定的模型（如豆包、Qwen）。
1. **尝试国产新特性：** 豆包的“专家模式”和 Kimi k2.5 在特定垂直领域（如媒体策划、视觉设计）表现出色，值得一试。
1. **税务提醒：** 有群友提到，境外收入超过 10W 人民币通常需要申报纳税。
