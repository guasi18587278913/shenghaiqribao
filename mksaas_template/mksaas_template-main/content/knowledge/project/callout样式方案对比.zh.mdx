# Callout 样式方案对比测试

下面展示10种不同风格的高级 Callout 设计方案,每种方案都有其独特的设计理念和视觉效果。

---

## 方案 1: GitHub 风格 - 简约专业

<div className="callout-style-1">
  <div className="callout-info">
    <strong>GitHub Style</strong>
    <p>简约、专业、清晰的设计风格。使用浅色背景和细边框,强调内容而非装饰。适合技术文档和开发者社区。</p>
    <ul>
      <li>简洁的视觉语言</li>
      <li>优秀的可读性</li>
      <li>专业的技术氛围</li>
    </ul>
  </div>
</div>

---

## 方案 2: Stripe 风格 - 优雅渐变

<div className="callout-style-2">
  <div className="callout-info">
    <strong>Stripe Style</strong>
    <p>使用微妙的渐变和阴影,创造出深度和层次感。边框和背景都使用渐变效果,显得更加精致和现代。</p>
    <ul>
      <li>精致的渐变效果</li>
      <li>立体的视觉层次</li>
      <li>现代商务风格</li>
    </ul>
  </div>
</div>

---

## 方案 3: Notion 风格 - 温暖柔和

<div className="callout-style-3">
  <div className="callout-info">
    <strong>Notion Style</strong>
    <p>温暖的配色,柔和的圆角,创造出友好亲切的氛围。使用emoji或图标增加趣味性,适合知识库和笔记应用。</p>
    <ul>
      <li>友好的视觉风格</li>
      <li>丰富的色彩选择</li>
      <li>灵活的内容组织</li>
    </ul>
  </div>
</div>

---

## 方案 4: Linear 风格 - 极简主义

<div className="callout-style-4">
  <div className="callout-info">
    <strong>Linear Style</strong>
    <p>极简设计,去除一切多余元素。使用细线和留白创造优雅的视觉效果,强调内容本身的重要性。</p>
    <ul>
      <li>极致的简约美学</li>
      <li>精确的排版控制</li>
      <li>优雅的留白设计</li>
    </ul>
  </div>
</div>

## 方案 5: Vercel 风格 - 高对比度

<div className="callout-style-5">
  <div className="callout-info">
    <strong>Vercel Style</strong>
    <p>使用高对比度的黑白灰色系,配合精心设计的间距和字重。创造出清晰、现代、专业的视觉效果。</p>
    <ul>
      <li>清晰的视觉层次</li>
      <li>强烈的对比效果</li>
      <li>现代科技感</li>
    </ul>
  </div>
</div>

---

## 方案 6: Tailwind 风格 - 色彩丰富

<div className="callout-style-6">
  <div className="callout-info">
    <strong>Tailwind Style</strong>
    <p>丰富的色彩系统,每种类型都有独特的配色。使用饱和度适中的颜色,既醒目又不刺眼。</p>
    <ul>
      <li>丰富的色彩体系</li>
      <li>平衡的饱和度</li>
      <li>清晰的类型区分</li>
    </ul>
  </div>
</div>

## 方案 7: Apple 风格 - 玻璃拟态

<div className="callout-style-7">
  <div className="callout-info">
    <strong>Apple Style</strong>
    <p>使用毛玻璃效果(Glassmorphism)和精致的阴影,创造出透明、轻盈的视觉效果。现代而富有科技感。</p>
    <ul>
      <li>透明的玻璃效果</li>
      <li>精致的模糊背景</li>
      <li>轻盈的视觉感受</li>
    </ul>
  </div>
</div>

---

## 方案 8: Discord 风格 - 深色主题

<div className="callout-style-8">
  <div className="callout-info">
    <strong>Discord Style</strong>
    <p>为深色主题优化的设计,使用低饱和度的颜色和柔和的发光效果。适合长时间阅读和夜间使用。</p>
    <ul>
      <li>护眼的深色配色</li>
      <li>柔和的发光效果</li>
      <li>舒适的阅读体验</li>
    </ul>
  </div>
</div>

---

## 方案 9: Medium 风格 - 编辑优先

<div className="callout-style-9">
  <div className="callout-info">
    <strong>Medium Style</strong>
    <p>以内容为中心的设计,使用衬线字体和宽松的行距。创造出类似印刷品的高品质阅读体验。</p>
    <ul>
      <li>优雅的衬线字体</li>
      <li>舒适的行距设置</li>
      <li>印刷品质感</li>
    </ul>
  </div>
</div>

---

## 方案 10: Figma 风格 - 创意设计

<div className="callout-style-10">

    <ul>
      <li>大胆的色彩运用</li>
      <li>创意的装饰元素</li>
      <li>生动的视觉表达</li>
    </ul>
