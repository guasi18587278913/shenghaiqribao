想法跑得比钱包快？先用好下列策略：免费额度薅到底、合购+月付、预算留给真正刚需。

---

## 1. 免费额度与薅羊毛
- **模型/工具**：Gemini、CodeBuddy（免费 GPT‑4）、Claude Web（有限制但够学习）、Bolt 每日 30k tokens、新注册 Comet 邀请送 1 个月 Pro。
- **玩法**：多个账号轮流用免费配额（Zzz 的做法）；不同平台互补，别把所有需求压在一个工具上。

## 2. 合购/拼车
- **V0 账号**：闲鱼“海鲜市场”上一年不到 200 元（糖饼实测），适合新手短期使用。

<Callout type="warning">
- **GPT Team**：可以多人拼，但存在风控风险，谨慎使用。
</Callout>

## 3. 订阅节奏
- 先用免费额度体验，确认刚需再付费；绝多数工具提供月付或试用。

<Callout type="warning">
- Cursor 官方支持退款（Never 亲测）；不要一开始就买年费。
</Callout>
- 养成“到期前复盘”习惯：继续用？降级？换平台？

## 4. 预算提醒
- Claude Code 深度使用远超 20 美金/月，大娄反馈“套餐不够用”；需要提前准备预算或选择中转模式。
- Cursor 可以从 PDD 月号 38 元起步，正式渠道再升级。
- “把钱花在刀刃上” = 工具链稳定 > 一堆会员；能用免费版搞定的坚决不付费。

> 省下来的每一笔钱，都能投入到更关键的事情上：买域名、拉真实流量、或者犒劳自己继续坚持下去。
