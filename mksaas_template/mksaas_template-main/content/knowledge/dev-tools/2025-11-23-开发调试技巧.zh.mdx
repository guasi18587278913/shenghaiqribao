* **环境变量排查：** 遇到 Auth 认证失败（如 GitHub/Google 登录报错），除了肉眼检查，建议在后端代码中使用 `console.log` 打印相关环境变量。很多时候是复制粘贴时少了一个字符或多了空格导致肉眼难以发现。

* **API 价格说明：** 关于 Nano Banana 模型 API 价格波动，助教澄清：早期价格为官方未定价时的自行补贴价，官方定价出炉后已同步调整，学员仍可联系助教申请 VIP 优惠。
