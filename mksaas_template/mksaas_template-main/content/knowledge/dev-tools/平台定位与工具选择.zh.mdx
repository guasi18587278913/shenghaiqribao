| 平台 | 适用场景 | 优势 | 局限 |
| --- | --- | --- | --- |
| **V0 (Vercel)** | 新手快速做原型 | 免费额度慷慨、UI 漂亮、部署直达 Vercel、一次成功率高 | 免费版导出受限，高级功能需付费 |
| **Bolt.new** | 愿意付费换效率 | 生成速度快、支持 Next.js | 免费额度极少、部署常遇网络报错、成本高 |
| **Lovable** | 需要高颜值展示页 | 视觉最佳、免费额度尚可 | 不支持 Next.js，一次成功率略低 |
| **进阶组合** | Cursor + Claude Code/Codex | 深度迭代、可控性强 | 对环境要求高、学习成本更高 |

使用策略（SuperEric & 多位学员总结）：

- 
<Callout type="info">
**学习期**：V0 免费额度足够练习；把流程跑通最重要。
</Callout>

- **追求颜值**：Lovable 适合做展示型产品，再导出到 Cursor 打磨逻辑。

- **预算充足/要速度**：Bolt 付费版快，但免费用户很快触顶。

- **进入实战**：把原型导出到 Cursor，由 Cursor + Claude Code 才能稳定迭代。
