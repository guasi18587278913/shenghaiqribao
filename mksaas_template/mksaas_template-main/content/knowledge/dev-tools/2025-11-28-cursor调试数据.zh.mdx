有群友疑惑为何 Cursor 无法自动修复 Vercel 部署或数据库写入错误。

* **原因分析：** Cursor 本身无法直接“看到”外部环境（如 Vercel 后台日志或数据库表结构），除非通过 MCP（Model Context Protocol）协议进行连接。
