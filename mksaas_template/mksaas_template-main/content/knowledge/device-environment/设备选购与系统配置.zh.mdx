## 1. Windows 方案（能用，但要多折腾）
- **最低配置**：16GB 内存 + SSD（≥512GB）+ i5/Ryzen5 级别 CPU，最好是 2018 年后的机器。
- **优先处理事项**：
  1. 梯子可用、节点纯净；
  2. Google 三件套完整；
  3. 确认磁盘空间足够，必要时迁移微信数据、加装硬盘或重装系统。

<Callout type="info">
- **WSL**：理论上可帮你用 Linux 工具，但代理配置极其复杂，多位学员折腾 2–3 天仍报错，推荐新手直接用 PowerShell + 原生环境。
</Callout>

<Callout type="info">
- **Cursor/Claude Code**：都支持 Windows，但需要额外配置 TUN、代理、依赖，体验不如 Mac。建议：用 Windows 跑完整套流程、确定长期投入后再升级。
</Callout>

## 2. Mac 选型指南

<Callout type="info">
| 场景 | 推荐配置 | 参考价格（2024Q4） | 备注 |
</Callout>
| --- | --- | --- | --- |
| **预算最紧** | Mac mini M4 · 16–24GB RAM · 512GB SSD | 咸鱼/京东教育优惠 2300–3000 元 | 性价比最高，需自备显示器；可远程控制使用 |
| **需要便携** | MacBook Air M3/M4 · 16–24GB RAM · 512GB | 教育优惠约 7000–8000 元 | 绝大多数 API 开发场景够用 |
| **一步到位** | MacBook Pro M4 Pro · ≥24GB RAM · 512GB+ | 1 万元以上 | 更好的散热和接口，适合长期重度开发 |
| **顶配土豪** | M4 Max · 48–96GB RAM · 1–2TB SSD | 2–4 万元 | 可跑本地大模型，但对 API 场景来说过剩 |

**内存选择**：
- 16GB 能用但会“爆内存”；
- 24GB 体验最佳；
- 32GB+ 对重度多任务更友好（Mini/Pro 选购时一次到位，后续无法升级）。

**硬盘**：
- 512GB 对大多数人足够，代码/依赖占用不大；
- 可搭配移动 SSD 或 NAS 存储素材、备份。

**二手与入手时机**：

<Callout type="warning">
- 咸鱼/转转挑选 M 系列，避免 Intel；查看电池循环、外观、是否升级到最新系统。
</Callout>
- 等新品发布时购买上代（“等 M5 出了买 M4”）能省 20–30%。
- Apple 教育优惠、京东百亿补贴、官翻/二手都可重点关注。

<Callout type="info">
**实用建议**：
</Callout>
- 先用现有设备 → 做出成果 → 用收入再升级；
- 若暂时买不起，可考虑按天租 Mac，逼自己产出。

## 3. 硬件外设与搭配

- **显示器**：Mac mini 用户必备，27" 2K 起步，预算足可上 4K；重点看色彩和接口。

<Callout type="warning">
- **键鼠**：罗技系口碑最好；苹果原装鼠标口碑差。Mac 版键盘能避免快捷键混乱。
</Callout>
- **扩展坞**：MacBook 多接口需求必备，确保支持供电 + HDMI。
- **椅子/桌面**：长时间编程别忽视人体工学，护腰比换电脑划算。

<Callout type="info">
- **NAS/移动硬盘**：大娄/Allan 强烈推荐，用 NAS 存储素材、备份，可跨设备无缝切换。
</Callout>

---

## 4. 手机与辅助设备

<Callout type="info">
- **Google Play**：建议备一台原生 Android（Pixel 4 以上，600–800 元二手）或自带 Google 套件的小米/一加机型。华为鸿蒙不可用。
</Callout>
- **远程控制**：Windows 笔记本远程 Mac mini、iPad Sidecar 等都是常见组合。

## 5. Mac 环境初始设置

- 首次运行第三方应用时，前往 `系统设置 > 安全性与隐私 > 通用`，允许“仍要打开”。
- 如果需要显示“任何来源”选项，可运行：
  ```bash
  sudo spctl --master-disable
  ```bash
- 其余开发工具（Homebrew、Node、Git、VSCode/Cursor 等）按照官方文档安装即可，Mac 平台基本一步到位。

> 结论：工具不是目的。先用得顺手，再谈升级；先保养好腰、眼睛和网络，再去追更贵的设备。
