适用对象：计划直接使用官方 Claude Web、Claude Code、Claude App 的成员。该服务对环境与身份的风控最严，务必在动手前把网络、手机号、支付方式全部准备好。

<Callout type="warning" title="风险与前置条件">
- **封号几乎不可逆**：志愿者欣霖、SuperEric 多次提醒，一旦环境异常会立即封禁，只能重新注册。
- **纯净静态住宅 IP 是底线**：注册前先用 `[ping0.cc](https://ping0.cc/`、`https://scamalytics.com/`) 检测风险值；Medium/High Risk 的节点一律换掉。
</Callout>

<Callout type="warning">
- **环境隔离**：同一套 IP、浏览器指纹、操作系统只绑定一个账号。不同账号要在虚拟机/不同用户下隔离，避免串号。
</Callout>

### 手机号与账号准备
- 国内手机号在 Claude 场景下几乎必封，需提前准备海外号码。
- 节点国家要与手机号国家匹配；未匹配会触发验证或封号。
- 注册 Claude 前请先准备好 Gmail/Google Play 生态账号，后续订阅也要用到。

#### 常用手机号方案
- **giffgaff 英国 SIM（最稳妥）**
  - 购买：拼多多 6–20 元、闲鱼约 20 元，记得选择空卡自行激活。
  - 激活：首次充值 10 英镑，带邀请码到账 15–25 英镑，支付用纯 VISA/Mastercard。
  - 保号：每 180 天跑一次流量或发短信即可；10 英镑余额理论上能用 5 年以上。
  - 功能：可接收短信、拨打电话、支持 WiFi Calling。体验贴近真实手机号，适合长期使用。
- **摩尔多瓦 eSIM**
  - 成本：40 列伊（约 16 元），无需实体卡、无需实名。
  - 操作：选择 +373 号段 → 购买 40 列伊套餐 → 在线开通 → 即可收短信。

<Callout type="info">
注意：需要自己在官网开通，不要找中间商代办以防诈骗。
</Callout>
- **美国 Purple Card（紫卡）**
  - 成本：约 3 美元/月，有月租。
  - 优势：为美国节点准备的稳定手机号，可搭配美区网络环境。
  - 注意：需按月续费，记得设置提醒。
- **接码平台（一次性验证）**
  - 代表：PingMe、freephone.io、SMS-Activate。
  - 成本：几元到十几元一次；号码可能被反复使用，适合练手或补救。
  - 操作要点：国家与梯子一致；验证码未到就换号；成功率非 100%，要准备多试几次。
- **成品号 / 中转号**
  - 淘宝、闲鱼有已经注册好的 Claude 成品号；群友反馈有稳定用几个月的案例。
  - 购买后第一时间改密码、绑定自己的邮箱，并配合纯净 IP 使用。
  - 也可使用春秋哥中转（`[aicodewith.com](https://aicodewith.com/?invitation=UFAV6EW`）、AnyRouter（`https://anyrouter.top/register?aff=KOmR`，每日签到送) 25 美金额度）等可靠中转。

#### 环境组合成功案例
- @Ethan：使用 V2free（中转）+ Kookeey 静态住宅 IP + Clash Verge，直接用 Google 账号注册，无需手机号验证。
- 多位同学反馈：选择美国住宅 IP + 当地实体 SIM，注册成功率最高；香港节点常被拒。

### 正式注册流程

<Callout type="warning">
1. 连接风险值 &lt;30 的纯净节点，确认浏览器、终端都走同一代理。
</Callout>

<Callout type="info">
2. 用准备好的 Gmail 访问 Claude 官网，按提示填写信息。
</Callout>
3. 进行手机号验证（或使用下方绕过方案）；完成后保存欢迎邮件。
4. 首次登录后，在同一环境下完成基础使用测试，再进入绑定支付或安装 Claude Code 等步骤。

### ⚡ 电话验证绕过方案（助教壹塵分享）
1. 在 Google Play 安装 Claude App，并用同一个 Google 账号登录。
2. Google 侧完成 OAuth 授权后返回 App，即可绕过电话号码验证。
3. 仍需保证使用纯净 IP；若要订阅，在 Google Play 中绑定海外信用卡即可。

### 常见问题排查
- **401 Invalid bearer token**  
  场景：网页可用，终端报错。  
  处理：让 Claude 自查配置文件，删除旧的 `anthropic_token` 路径；若仍失败，换到日本等成功率更高的节点。
- **频繁封号 / 登录异常**  

<Callout type="warning">
处理：被封后无法解封，只能重新注册并更换谷歌账号。保持节点稳定、避免短时间内大量 API 调用。
</Callout>
- **无法通过验证**  
  先检查节点是否为住宅 IP；必要时换客户端、清理浏览器缓存，再重新尝试。

### ⚡ 替代与过渡方案
- 纯净环境尚未搞定时，可使用国内套壳（例如 2233）体验功能，优先保持学习节奏。
- 初学阶段可先在 Cursor 内使用 Claude 模型，等产品成型或需要高级功能时再申请官方账号。
