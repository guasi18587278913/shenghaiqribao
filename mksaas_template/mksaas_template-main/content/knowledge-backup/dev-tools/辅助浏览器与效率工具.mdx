# 辅助浏览器与效率工具

### Dia 浏览器
- **硬件要求**：仅支持 Mac M 系列芯片；Intel Mac、Windows 无法运行。
- **注册与下载**：`https://www.diabrowser.com/`，当前无需邀请码。
- **中文体验**：安装 “沉浸式翻译” 插件；Personalization 中设置 “请使用简体中文作答”。
- **功能**：AI 自动搜索总结、截图提问、网页内容问答；书签支持 Pin/Bookmark 分类。
- **常见问题**：
  - 打不开应用 → 检查芯片（Intel 不支持）与系统版本（建议 macOS 13+），“隐私与安全性” 中选择 “仍要打开”。
  - Windows/Intel Mac 替代 → 豆包浏览器、Comet、Monica/Sider 插件等。
  - Dia 仅做辅助调研工具，翻译仍用沉浸式翻译效率更高。

### OpenAI ChatGPT Atlas
- 2024/10/21 发布，macOS（M 系列）专用 AI 浏览器，包含在 GPT 订阅内；与 ChatGPT 数据互通。
- 优势：原生整合，无需额外付费。劣势：仅 Mac、初期卡顿过度思考。

### Comet 浏览器（Windows 福音）
- 邀请链接注册可获 1 个月 Pro；可从 Chrome 无缝迁移标签和书签。
- 实测总结准确、风格理工；支持 agent 与 pplx，闲鱼年费约 200 元。

### 豆包浏览器
- 字节出品，Windows 平台首选的 Dia 平替；配合豆包桌面悬浮窗、QQ 浏览器等都可完成日常调研。

### 翻译与效率工具
- **沉浸式翻译**：Chrome 插件，完全免费；推荐翻译引擎 “Bing 翻译”，快捷键 `Alt + A`。
- **微信截图翻译**：Alt + A 截图 → 点击翻译。
- **Mac 工具**：Paste（剪贴板管理）、CleanShot（高级截图）。
- **调试工具**：Chrome DevTools（`F12`），Elements 选中元素替代 Stagewise。

### 其他调研资源
- Toolify、AIGC 排行榜、TAAFT 等 AI 导航。
- NotebookLM：谷歌推出的文档/视频总结工具，完全免费。
- Raindrop.io：跨平台书签管理。
