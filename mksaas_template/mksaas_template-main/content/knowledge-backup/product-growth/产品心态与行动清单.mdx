# 产品心态与行动清单

- **MVP 原则**：先做 20% 的核心功能并上线，真实用户反馈才是最好的产品经理。
- **记录与复盘**：把开发/推广过程输出到社媒、博客或群里，收获用户同时积累个人品牌。
- **组合拳**：痛点挖掘工具 + AI 竞品分析 + 快速原型 + Cursor 精修 + 高频迭代。
- **定期回顾**：每两周检查数据（访问、注册、转化），根据结果调策略，而不是盲目叠功能。

> “执行力碾压一切。问清楚‘什么人/什么场景/愿意花多少钱/解决什么问题’，剩下的就是不断做、不断试。”——来自多位助教与学长的共识。
