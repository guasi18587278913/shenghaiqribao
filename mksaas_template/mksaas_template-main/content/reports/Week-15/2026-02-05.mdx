### 1. 🛠️ 基建实操：OpenClaw 服务器选购与配置

针对 OpenClaw 的部署环境，群友们总结了一套高性价比且避坑的方案：

* **服务器选购避坑（重点）：**

  * **误区纠正：** 很多群友想买海外节点服务器，误以为必须去“腾讯云国际版”，导致卡在需要海外身份认证上。

  * **正确姿势：** 在**腾讯云国内版**直接购买，地域选择“海外地区”（如香港、新加坡等）即可，无需海外身份，且支持国内支付方式。

* **价格情报（薅羊毛）：**

  * **新用户福利：** 腾讯云针对新用户有大额优惠，4核4G配置可能低至 39元/月 或 99元/年（具体视活动而定）。

  * **常规高性价比：** 双核4G配置约为 199元/年。

  * **替代方案：** 也有群友选择本地部署，配合 Kimi API 或 Antigravity 使用，成本更低。

### 2. 💸 成本控制：Antigravity 与 Token 经济学

在解决算力之后，大家重点讨论了如何降低 AI 模型的调用成本：

* **Antigravity 工具探讨：**

  * **账号获取：** 闲鱼上 Antigravity Pro 年费约为 69.9元。更有群友分享“极限操作”：通过谷歌账号的学生认证（GitHub Student Pack 等渠道）获取资格，自行购买卡密仅需 5元左右，但需要一定动手能力。

  * **实用性评价：**

    * **正方：** 配合 OpenClaw 玩玩足够，能显著降低成本。

    * **反方：** Pro 版本对于高强度编程来说“根本不够用”，主力生产力依然建议使用 Codex 或正版 Claude/Gemini，尽管 Gemini 近期表现被吐槽“越来越不行”。

* **API 额度解析：**

  * Copilot/Codex 结合 Claude 被认为是目前最强的编程组合。

  * 关于 Claude 的额度，有观点认为购买 100刀的额度性价比最高，咸鱼上的小额转发（如69块1000次）适合尝鲜。

### 3. ⚙️ 技术疑难与工具反馈

* **反代配置问题：** 有群友尝试将 Antigravity 反代给 OpenClaw 使用时遇到困难，表现为重启后网关服务无法启动。建议检查配置文件或咨询相关文档。

* **OpenClaw 好评点：** 使用了几天的群友反馈该工具非常强大，特别是支持自己控制切换模型、切换供应商以及 Codex oauth 认证。

* **Pencil MCP 问题：** 有用户反馈 Pencil MCP 生成的图片右键无法导出，可能是工具本身的兼容性或浏览器限制。
