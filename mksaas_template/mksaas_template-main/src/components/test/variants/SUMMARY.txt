🎯 使用建议
================================================================================

【最优选择】推荐使用 方案D (SidebarDAutoCollapse)

理由:
✓ 空间利用最优 (同时最多展开1个分类)
✓ 用户交互清晰 (单一焦点)
✓ 自动收起避免混淆
✓ 长列表智能截断 (8篇限制)
✓ 性能最佳 (DOM节点数少)
✓ 适应性强 (适合各种场景)
✓ 综合体验最好

【备选方案】

- 需要多个分类同时展开 → 方案A
- 空间极其受限 → 方案B
- 喜欢悬停预览 → 方案C
- 对比参考 → 方案E

   <SidebarDAutoCollapse
     knowledgeCategories={categories}
     selectedCategory={selected}
     onSelectCategory={setSelected}
   />

   - 修改文章显示数: 改 ARTICLES_LIMIT 常量 (默认8)

4. 集成生产:
